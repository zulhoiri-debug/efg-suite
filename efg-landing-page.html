FAQ

Pertanyaan Umum

Apakah perlu install software?+
Tidak! Semua tools berjalan di browser (Chrome, Firefox, Edge). Cukup buka file HTML, tidak perlu install apapun.
Berapa biaya API key Gemini dan Clipdrop?+
Keduanya gratis! Gemini API gratis 1.500 request/hari. Clipdrop API gratis 100 upscale/hari. Cukup untuk penggunaan normal sehari-hari.
Apa itu license key dan bagaimana cara kerjanya?+
License key memastikan tools hanya bisa dipakai di jumlah browser sesuai paket yang dibeli. Key divalidasi secara lokal — tidak perlu koneksi internet untuk aktivasi setelah diaktifkan pertama kali.
Apakah tools bisa dipakai untuk semua jenis gambar?+
Ya! Tools mendukung foto (JPG, PNG, WEBP), ilustrasi, vektor (SVG, EPS, AI, PDF), dan video (MP4, MOV) untuk metadata generation.
Paket Agency boleh dijual ulang ke orang lain?+
Ya, paket Agency sudah termasuk hak resell. Kamu bisa ganti nama & logo (white label) lalu jual ke klienmu dengan harga sendiri.
Bagaimana cara mendapatkan update versi baru?+
Paket Pro dan Agency mendapat update gratis selamanya. File terbaru akan dikirim via WhatsApp saat ada update tersedia.
